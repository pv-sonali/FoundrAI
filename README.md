# 🚀 FoundrAI

### AI-Powered Startup Mentorship Platform

---

## 📌 Introduction

FoundrAI is an AI-powered startup mentorship platform designed to help aspiring entrepreneurs transform innovative ideas into successful businesses.

The platform acts as a **24/7 virtual startup advisor**, offering intelligent guidance across multiple stages of the startup journey including idea validation, market research, competitor analysis, MVP planning, business modeling, funding guidance, and investor pitch deck generation.

FoundrAI aims to make startup mentorship affordable, accessible, and scalable for founders worldwide.

---

# 🎯 Problem Statement

Many aspiring entrepreneurs face significant challenges while building startups due to lack of proper mentorship and guidance. Common problems include:

* Difficulty validating startup ideas
* Limited understanding of market demand
* Lack of competitor research
* Poor business model planning
* Unclear MVP development process
* Difficulty finding funding opportunities
* Inexperience in creating investor pitch decks

Traditional startup mentorship programs are often expensive and inaccessible for students and first-time founders, resulting in many startup failures before product launch.

---

# 💡 Proposed Solution

FoundrAI provides AI-driven mentorship and startup guidance through an intelligent and user-friendly platform.

Users can submit their startup idea, and the system generates:

* Idea validation analysis
* Market opportunity insights
* Competitor intelligence reports
* Business model recommendations
* MVP development roadmap
* Funding and investment suggestions
* Investor-ready pitch deck content

This enables entrepreneurs to make informed business decisions quickly and efficiently.

---

# ✨ Core Features

## 1. AI Idea Validator

Evaluates startup ideas based on feasibility, uniqueness, scalability, and market demand.

## 2. Competitor Intelligence

Identifies existing competitors, analyzes strengths and weaknesses, and highlights market opportunities.

## 3. Business Model Generator

Suggests monetization strategies, revenue models, and business frameworks.

## 4. MVP Roadmap Planner

Creates a structured roadmap for developing and launching a Minimum Viable Product (MVP).

## 5. Pitch Deck Generator

Automatically generates professional investor pitch deck content.

## 6. Funding Assistant

Recommends startup grants, incubators, accelerators, and potential investors.

## 7. AI Startup Mentor

Provides real-time startup guidance through an AI-powered chatbot assistant.

## 8. Risk Analyzer

Detects possible technical, financial, and market-related risks associated with the startup idea.

# ⚙️ Installation & Setup

## Clone the Repository

```bash
git clone https://github.com/your-username/foundrai.git
cd foundrai
```

## Install Frontend Dependencies

```bash
cd client
npm install
```

## Install Backend Dependencies

```bash
cd server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder and configure the following variables:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_jwt_secret
OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key
```

---

# ▶️ Running the Application

## Start Backend Server

```bash
cd server
npm run dev
```

## Start Frontend Application

```bash
cd client
npm start
```
